Nuestra página web esta fundada con el fin de ayudar al usuario que lo utilizará y  darle a conocer la manera de utilizarla generando un aprendizaje fácil y eficaz donde el niño comprenda cada cosa que esta realizando por si mismo, dando satisfacción al alumno con el afán de que sea interactivo y divertido para el usuario.
La aplicación web fue fundada con el afán de ayudar a los usuarios como niños aprender a leer y escribir de manera sencilla y de manera interactiva que el usuario interactue con la tecnologia con lo cual lo que se quiere lograr es: Hacer uso de las tecnologias de una manera responsable aplicandolo en diversas áreas de estudio.

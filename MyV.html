Mision


Nuestra pagina web esta fundada con el fin de ayudar al usuario que lo utilizara darle a conocer la manera de utilizarla generando un aprendizaje facil y eficaz donde el niño comprenda cada cosa que esta realizando por si mismo, dando satisfaccion a el alumno con el afan de que sea interactivo y divertido para el usuario.




Vision


La aplicacion web fue fundada con el afan de ayudar a los usuarios como niños aprender a leer y escribir de manera sencilla y de manera interactiva que el usuario interactue con la tecnologia con lo cual lo que se quiere lograr es: Hacer uso de las tecnologias de una manera responsable aplicandolo en diversas areas de estudio.
